Submission Checklist

  • Trained model / model checkpoints

    Model checkpoints should include all parameters required for inference.

  • Inference code

    Inference code should be directly executable on the official evaluation set.

  • Runtime environment configuration

    Provide a Dockerfile, environment.yml, installation script, or equivalent runtime setup.

  • System description document

    Describe model architecture, training data, inference procedure, parameter scale, and external resources used.

  • Inference dependencies

    Include tokenizers, vocoders, auxiliary files, and any other resources required to run inference.
